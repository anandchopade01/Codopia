What is CSS?

(Cascading style sheets)
It is used to control the visual presentation of a web page. CSS allows web developers to define the layout, colors, fonts, and other design elements of a webpage, separate from the content written in HTML. By using CSS, developers can create visually appealing and responsive websites that are consistent across different devices and screen sizes. CSS works by selecting HTML elements and applying styles to them using properties like color, font-size, margin, padding and more.
